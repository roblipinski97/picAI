# Public Institutions' Capacity Indicators - AI model

The code in this repository is part of the Public Institutions' Capacity (PIC) Indicators initative under EGVPI. It aims to use AI models to automatically collect answers to a set of survey-like items regarding the institutional ('de iure') arrangments of central governments around the world. At present, it focuses on Human Resources Management (HRM) and Public Financial Management (PFM) sectors.
